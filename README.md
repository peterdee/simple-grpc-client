## simple-grpc-client

A simple gRPC client with Node

### Deploy

```shell script
git clone https://github.com/peterdee/simple-grpc-client
cd ./simple-grpc-client
nvm use 14
npm i
```

### Environment variables

The `.env` file is required. See the [.env.example](.env.example) file for details.

### Launch

```shell script
npm start
```

### Linting

```shell script
npm run lint
```
